Ingress ICE

Ingress ICE is a timelapse maker for Ingress by capturing screenshots of Ingress Intel Map



Printed from http://nibogd.github.io/ingress-ice/


Features:

  • Captures screenshots of ingress intel map every n seconds
  • Set your location
  • Supports 2-step authentication
  • Doesn't require X server to be run.
  • [Linux only] Download and run ./make_video.sh to make a video.
